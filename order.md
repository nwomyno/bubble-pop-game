# 구현 단계

## 순서

1. 기본 화면 출력
    [] 실행 시 어두운 파란색 화면이 떠야 함.
2. 정적 버블 표시함
    [] 화면 상단에 육각형 패턴 버블망이 보이면 OK.
3. 발사대 조작
    [] 좌우 방향키로 발사대가 회전하면 성공.
4. 버블 발사, 이동
    [] 스페이스바로 버블이 발사되고, 벽 반사가 되면 성공.
5. 충돌, 부착
    [] 버블이 기존 격자에 정확하게 붙으면 성공.
6. 버블 터뜨림
    [] 같은 색 3개 연결 시 터지고 점수가 오르면 성공.
7. 게임 규칙 완성
    [] 4발마다 벽이 내려오고, 스테이지가 전환되면 성공.
8. UI, 마무리
    [] 점수 표시, 스테이지 정보, 다음 버블 미리보기, 종료 메시지가 나오면 완성.

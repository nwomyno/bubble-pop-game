- 게임 에셋 제작 (버블 등)

## 개발

### 건택 - 메인 개발

- 프로젝트 구조 설계, skeleton code 기반 클래스 구조 정의
- 시스템 아키텍처와 물리 엔진, 게임 진행 시스템 구현
- 기본 게임 로직, 기존 UI 요소를 기반으로 기능 추가 구현
- 버블의 표현 강화, 최소 기능(MVP)을 모두 구현
- 코드 품질 관리를 위해 docstring 작성, 로직에 줄 단위 주석 추가
- git으로 버전관리, 리포지토리를 구축하여 프로젝트 관리
